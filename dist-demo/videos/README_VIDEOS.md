## 🎬 Especificações Técnicas

### Formato Recomendado
- **Formato:** MP4 (H.264)
- **Resolução:** 1920x1080 (Full HD) ou 1280x720 (HD)
- **Aspect Ratio:** 16:9
- **Bitrate:** 2-5 Mbps
- **Áudio:** AAC, 128-192 kbps

### Tamanho
- Máximo 50 MB por vídeo (para carregamento rápido)
- Se maior, considere comprimir

---

## 📥 Como Adicionar os Vídeos

### Opção 1: Copiar Manualmente
1. Grave/edite seus vídeos
2. Exporte no formato MP4
3. Renomeie conforme a lista acima
4. Copie para: `C:\Users\Informatica\tubaraoemprestimo\public\videos\`

### Opção 2: Via Terminal
```bash
cd ~/tubaraoemprestimo/public/videos
# Copie seus vídeos para esta pasta
cp /caminho/dos/seus/videos/*.mp4 .
```
